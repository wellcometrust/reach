How Reach works

Reach uses machine learning to search through thousands of policy documents, extract the references and match them with scientific publications.

1. Download

A piece of software called a ‘scraper’ is used to extract documents from the websites of policy organisations. The scraper navigates through a website and clicks on all the links. Where a link leads to a PDF, it is downloaded and the text is extracted. You can browse our collection of 129k policy documents here.

2. Extract

An algorithm scans the text from each PDF and identifies sections that contain references. It does this by identifying headings such as ‘References’, ‘Bibliography’ or ‘End Notes’. Individual references are extracted from the sections when the algorithm identifies the start and the end of each reference. The references are then split into individual parts like author, publication date, journal and title.

3. Match

Once the parts of each reference have been extracted, they are compared with a database of scientific publications. References with the same title as a publication in the database are matched. This checks that the algorithm has identified scientific publications from the downloaded PDFs.

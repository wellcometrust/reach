About Reach

Reach was developed to help researchers, policy makers and funders to better understand how scientific research is used in policy.

Open Source

We are committed to making Reach an open source tool. Our entire code can be found on GitHub and our open API gives you access to the data that powers Reach.

Accuracy

Reach is powered by cutting edge machine learning techniques applied by the data scientists in Wellcome Data Labs. Machine learning is not 100% accurate because it is based on probability, but we are constantly working on Reach to improve its accuracy. How Reach works

Reach uses machine learning to search through thousands of policy documents, extract the references and match them with scientific publications.

1. Download

Reach downloads PDFs from the websites of leading health policy organisations.

2. Extract

An algorithm is used to extract references from the PDFs.

3. Match

References are verified using a database of scientific publications.
